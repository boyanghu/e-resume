Hydrate Now Vancouver

This is a web application made using the Google Web Toolkit (GWT). The client asked for an intuitive website that geo-located all of the drinking fountains in the metro-Vancouver. For this project my team and I decided to follow the Scrum development process complete with bi-weekly sprints, user stories, daily stand-ups, and retrospectives. This site primarily target bikers and pedistrians who needed re-hydrate, which is why we implemented two clever search capabilities to find the closest drinking fountain to any point of interest.
